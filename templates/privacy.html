1. Сбор персональных данных

Мы собираем только те данные, которые необходимы для предоставления наших услуг и улучшения пользовательского опыта.

1.1. Данные, которые мы собираем
  • Контактная информация: имя, email, телефон
  • Профиль пользователя: предпочтения, история бронирований
  • Технические данные: IP-адрес, тип браузера, время посещения
  • Данные о поездках: даты, направления, количество человек
1.2. Автоматический сбор

Некоторые данные собираются автоматически при использовании нашего сайта для аналитики и улучшения сервиса.

2. Использование персональных данных

2.1. Основные цели использования
  • Обработка и подтверждение бронирований
  • Предоставление информации об экскурсиях
  • Связь с клиентами по вопросам поездок
  • Улучшение качества наших услуг
  • Персонализация предложений и рекомендаций
2.2. Маркетинговые цели

С вашего согласия мы можем использовать ваши данные для отправки информации о новых экскурсиях и специальных предложениях.

3. Передача персональных данных

3.1. Когда мы передаем данные

Мы не продаем, не обмениваем и не передаем ваши персональные данные третьим лицам, за исключением случаев, описанных ниже.

3.2. Случаи передачи данных
  • Партнеры по экскурсиям: гиды, отели, транспортные компании
  • Платежные системы: для обработки платежей
  • Правоохранительные органы: по требованию закона
  • С вашего согласия: для дополнительных услуг
3.3. Защита при передаче

Все передачи данных осуществляются с использованием защищенных соединений и протоколов.

4. Безопасность данных

4.1. Меры защиты
  • Шифрование данных при передаче (HTTPS/SSL)
  • Защищенные серверы с ограниченным доступом
  • Регулярные обновления систем безопасности
  • Мониторинг и обнаружение угроз
  • Резервное копирование данных
4.2. Обучение персонала

Наши сотрудники проходят регулярное обучение по вопросам защиты персональных данных.

5. Ваши права

5.1. Права на данные
  • Право доступа: получить информацию о том, какие данные мы храним
  • Право исправления: исправить неточные данные
  • Право удаления: удалить ваши данные (с ограничениями)
  • Право ограничения: ограничить обработку данных
  • Право переноса: получить данные в структурированном виде
5.2. Как реализовать права

Для реализации ваших прав свяжитесь с нами через форму обратной связи или по email.

6. Файлы cookie

6.1. Что такое cookie

Cookie - это небольшие текстовые файлы, которые сохраняются на вашем устройстве для улучшения работы сайта.

6.2. Типы cookie
  • Необходимые: для базовой работы сайта
  • Функциональные: для запоминания ваших предпочтений
  • Аналитические: для анализа посещений и улучшения сервиса
  • Маркетинговые: для показа релевантной рекламы
6.3. Управление cookie

Вы можете управлять настройками cookie в вашем браузере или отказаться от их использования.
